## Wiki.js 좌측 네비게이션 색상

Admin → Theme / CSS 커스텀에서 사이드바 배경·텍스트 색 변경.

컨테이너 내 정적 CSS를 직접 수정하는 방법도 가능하나 업그레이드 시 덮어쓰기 주의.
