## 계정

```bash
useradd -d /home/계정명 계정명
passwd 계정명
chmod -R 701 /home/계정명
chown -R 계정명:계정명 /home/계정명
```

## Apache

1. `conf/httpd.conf` — `Listen` 포트
1. `conf.d/httpd-vhost.conf` — VirtualHost 추가
1. Apache 재시작

## JSP (Apache + Tomcat)

1. Apache `workers.properties` — worker·포트 추가
1. Tomcat `server.xml` — 동일 포트 connector
1. `mkdir -p Catalina+포트/localhost`
1. Apache·Tomcat 재시작
