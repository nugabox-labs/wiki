일정 기간 지난 파일·디렉터리 삭제.

```bash
#!/bin/bash

## 파일 (91일 초과)
find /경로 -type f -ctime +91 -exec rm {} ';'

## 디렉터리 (181일 초과)
find /경로 -type d -ctime +181 -exec rm -rf {} ';'
```
