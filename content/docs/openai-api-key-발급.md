## 발급 절차 (실제 키 본문 미포함)

1. [https://openai.com](https://openai.com/) → Login → API platform
1. Dashboard → API keys
1. Create new secret key → Name → Create
1. 키는 한 번만 표시 → 안전하게 보관 (외부 공유 금지)

## 참고

- 사용량 과금 / 결제수단 미등록 시 제한 가능
- 본문에 실제 secret key 없음 → Post 미적용
