## gitmoji-cli

```bash
npm i -g gitmoji-cli

git add .
gitmoji -c   # 이모지 선택 후 커밋 메시지 작성
```

- JetBrains IDE(IntelliJ, PyCharm, PHPStorm 등): 설정 > 플러그인 > gitmoji plus 설치 시 커밋 화면에서 바로 사용 가능
