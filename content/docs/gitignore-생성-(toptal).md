## gitignore 생성

- [https://www.toptal.com/developers/gitignore](https://www.toptal.com/developers/gitignore) — IDE/OS 조합 선택 후 생성
- 저장소 루트(`.git` 있는 폴더)에 `.gitignore`로 저장

## 예시 템플릿

PhpStorm, IntelliJ, VS Code, Eclipse, macOS, Windows, Adobe, PyCharm 등 조합:

```javascript
https://www.toptal.com/developers/gitignore?templates=phpstorm,intellij,visualstudiocode,eclipse,macos,windows
```
