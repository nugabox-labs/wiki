## 맥 텍스트 편집기(TextEdit)를 메모장처럼 바로 새 문서로 열기

```bash
defaults write -g NSShowAppCentricOpenPanelInsteadOfUntitledFile -bool false
```

기본값은 실행 시 파일 열기 패널이 뜨는데, 위 명령으로 바로 새 문서가 열리도록 변경
