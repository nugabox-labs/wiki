## Tibero Tablespace

### 버전

```sql
SELECT * FROM v$version WHERE NAME = 'TB_MAJOR' OR NAME = 'TB_MINOR';
```

### 사용량 조회

```sql
SELECT A.TABLESPACE_NAME, A.FILE_ID,
       ROUND(A.BYTES/1024/1024/1024, 2)||'G' USABLE,
       ROUND(B.BYTES/1024/1024/1024, 2)||'G' FREE,
       TRUNC(100-(B.BYTES/A.BYTES*100), 2)||'%' USED_RATE,
       TRUNC(B.BYTES/A.BYTES*100, 2)||'%' FREE_RATE
  FROM (SELECT TABLESPACE_NAME, FILE_ID, SUM(BYTES) BYTES
          FROM DBA_DATA_FILES
         GROUP BY TABLESPACE_NAME, FILE_ID) A,
       (SELECT TABLESPACE_NAME, FILE_ID, SUM(BYTES) BYTES
          FROM DBA_FREE_SPACE
         GROUP BY TABLESPACE_NAME, FILE_ID) B
 WHERE A.TABLESPACE_NAME = B.TABLESPACE_NAME
   AND A.FILE_ID = B.FILE_ID(+)
 ORDER BY USED_RATE DESC;
```

### AUTOEXTEND

```sql
SELECT TABLESPACE_NAME, FILE_NAME, AUTOEXTENSIBLE FROM DBA_DATA_FILES;

ALTER DATABASE DATAFILE '데이터파일경로' AUTOEXTEND ON NEXT 사이즈;
ALTER DATABASE DATAFILE '/tmax/tibero6/tbdata/TS_PLATFORM2018_01.dtf'
  AUTOEXTEND ON NEXT 104857600 MAXSIZE 21474836480;
```

### 생성·삭제

```sql
CREATE TABLESPACE TS_PLATFORM2018 DATAFILE
  '/tmax/tibero6/tbdata/TS_PLATFORM2018_01.dtf' SIZE 10737418240
  AUTOEXTEND ON NEXT 104857600 MAXSIZE 21474836480
LOGGING
ONLINE
PERMANENT
EXTENT MANAGEMENT LOCAL AUTOALLOCATE;

DROP TABLESPACE TS_PLATFORM2018 INCLUDING CONTENTS AND DATAFILES;
```
