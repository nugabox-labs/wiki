## 네트워크 연결 단계별 확인

**1) 물리적으로 연결돼 있는가 (랜 케이블)**

```bash
ethtool em1        # Link detected: yes → 케이블 연결됨 / no → 연결 안 됨
```

**2) 네트워크 인터페이스가 살아있는가**

```bash
ifconfig em1       # RX/TX bytes가 0이 아니면 활성 상태
```

**3) 로컬 네트워크(게이트웨이)에 있는가**

```bash
route -n            # -n: IP를 호스트명으로 변환 안 함, default 줄에서 게이트웨이 확인
```

**4) DNS가 동작하는가**

```bash
nslookup daum.net
```

**5) 원격 호스트로 경로 추적**

```bash
apt-get install tcptraceroute
traceroute daum.net   # 또는 tcptraceroute
```

**6) 원격 포트가 열려 있는가**

```bash
telnet 192.168.1.19 22     # 연결되면 배너 응답으로 확인

yum install nmap
nmap -p 80 daum.net         # open / closed(서비스 미실행) / filtered(방화벽 차단, 상태 불명)
```

**7) 리스닝/연결 포트 조회**

```bash
netstat -lnp | grep 80      # -l 리스닝만, -n IP 그대로, -p PID/프로그램명
netstat -np | grep 80       # 현재 연결 상태 조회 (ESTABLISHED 등)
```
