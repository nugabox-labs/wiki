## php.ini / php 실행파일 위치 확인

```bash
which php                        # php 실행파일 위치 (보통 /usr/bin/php)
php --ini | grep php.ini         # php.ini 경로 (보통 /etc/php.ini, 환경에 따라 다를 수 있음)
```
