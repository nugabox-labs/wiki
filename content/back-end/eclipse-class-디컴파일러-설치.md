> 원본이 스크린샷만 있어 텍스트 추출 불가. 일반 참고용 요약.

1. Help → Eclipse Marketplace → **Enhanced Class Decompiler** 설치 (JD-Core, Fernflower, CFR, Procyon 등)
1. 재시작
1. Window → Preferences → Java → Decompiler에서 엔진 선택
1. 소스 없는 `.class`(jar 내부 등) 더블클릭 시 디컴파일 소스 표시
