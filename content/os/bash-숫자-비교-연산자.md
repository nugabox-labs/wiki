## Bash 숫자 비교

```bash
[ "$a" -eq "$b" ]  # equal
[ "$a" -ne "$b" ]  # not equal
[ "$a" -lt "$b" ]  # less than
[ "$a" -le "$b" ]  # less or equal
[ "$a" -gt "$b" ]  # greater than
[ "$a" -ge "$b" ]  # greater or equal
```
