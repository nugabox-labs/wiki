## 쉘 메시지 영어(LANG)

한글 깨짐 → `LANG=en_US.UTF-8`

```bash
echo $LANG
LANG=en_US.UTF-8          # 현재 세션만
# 영구: ~/.bashrc 끝에
export LANG=en_US.UTF-8
```
