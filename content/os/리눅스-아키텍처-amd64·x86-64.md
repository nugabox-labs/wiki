## 리눅스 아키텍처 이름 정리

**AMD64 ≠ AMD CPU 전용.** AMD가 만든 x86-64 ISA이고, Intel도 라이선스(EM64T/Intel64)로 동일 계열.

| 표기 | 의미 |
| --- | --- |
| `i386` / `i686` / `x86` / `IA-32` | 32비트 x86 |
| `amd64` / `x86_64` / `x64` | 64비트 x86-64 |
| `IA-64` | Itanium (x86과 비호환, 별개) |

```bash
uname -m          # x86_64, aarch64 등
dpkg --print-architecture   # amd64, arm64 (Debian/Ubuntu)
```

요약: **32비트 = i386/x86**, **일반 PC 64비트 = amd64/x86\_64** (Intel·AMD 공통).
