## macOS 네트워크 .DS\_Store 생성 방지

```bash
defaults write com.apple.desktopservices DSDontWriteNetworkStores true
# 되돌리기: false
```

재부팅 후 적용.

## 기존 파일 삭제

```bash
find . -name '*.DS_Store' -type f -delete
```
