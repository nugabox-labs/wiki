## 시간·타임존 (CentOS)

```bash
clock          # H/W
date           # System
date MMDDhhmmYYYY.ss
clock -w       # System → H/W
rdate -s time.bora.net
echo 'rdate -s time.bora.net' >> /etc/rc.d/rc.local
ln -sf /usr/share/zoneinfo/Asia/Seoul /etc/localtime
# CentOS 7+
timedatectl set-timezone Asia/Seoul
chronyc tracking
```
