## VMware Fusion Pro 개인용(무료) 설치 (Apple Silicon)

1. Broadcom 계정 가입/로그인 후 VMware Fusion 다운로드 페이지에서 dmg 다운로드 ([support.broadcom.com](http://support.broadcom.com/) > VMware Fusion)
1. dmg 마운트 후 설치 아이콘 더블클릭
1. 라이선스 키 입력 화면에서 "개인 사용(라이선스 없이 사용)" 옵션 선택 후 계속
1. Windows VM 생성 시 "Get Windows from Microsoft" 선택 → Apple Silicon + Windows 11 ARM 옵션으로 진행
1. UEFI 옵션은 기본값 유지, Finish로 VM 생성 → 바로 Windows 설치 진행
