## 네트워크 드라이브에 .DS\_Store 생성 안 하게 하기

```bash
defaults write com.apple.desktopservices DSDontWriteNetworkStores true
# 되돌리기: true → false
```

재부팅하면 이후로 생성 안 됨

## 기존 .DS\_Store 찾기/삭제

```bash
find . -name '*.DS_Store'                    # 찾기
find . -name '*.DS_Store' -type f -delete    # 찾아서 삭제
```
