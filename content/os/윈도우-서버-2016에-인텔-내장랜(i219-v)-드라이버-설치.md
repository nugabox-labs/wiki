## Windows Server 2016에 인텔 I219-V 랜카드 드라이버 설치

- 문제: I219-V가 인텔 공식 드라이버 지원 목록에 없어 인식 안 됨

**1) 서명 검사 비활성화 (설치 전)**

```shell
bcdedit -set loadoptions DISABLE_INTEGRITY_CHECKS
bcdedit -set TESTSIGNING ON
```

재부팅.

**2) 드라이버 다운로드 및 inf 수정**

- [Intel Network Adapter Driver for Windows Server 2016](https://downloadcenter.intel.com/download/26092/) 다운로드 (exe, 반디집 등으로 압축 해제 가능)
- `\PRO1000\Winx64\NDIS65\e1d65x64.inf` 파일 열기
- `[ControlFlags]` 섹션의 내용을 삭제
- `[Intel.NTamd64.6.2.1]` 섹션에서 `%E15B8NC`(I219-V) 관련 3줄을 찾아 `[Intel.NTamd64.6.2]` 섹션에 복사
- 저장

**3) 드라이버 설치**

- 장치 관리자 > Ethernet 컨트롤러 우클릭 > 드라이버 소프트웨어 업데이트 > 컴퓨터에서 드라이버 찾아보기 > 수정한 inf가 있는 경로 지정(하위 폴더 포함) > 보안 경고에서 설치 진행

**4) 서명 검사 원복**

```shell
bcdedit -set loadoptions ENABLE_INTEGRITY_CHECKS
bcdedit -set TESTSIGNING OFF
```

재부팅.
