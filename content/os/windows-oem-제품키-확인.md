## Windows OEM 제품키 확인 (에이텍 BIOS)

### Windows 7

1. `run.exe` → `C:\` 복사
1. 관리자 CMD → `CD\` → `run` → productkey 확인

### Windows 10/11

1. `OEM produkey KEY Confirmation.bat` 저장
1. 관리자 CMD 실행 → productkey 확인
