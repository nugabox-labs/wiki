## 인앱 브라우저 → 외부 브라우저 (웹 UX 스킴)

메신저 WebView에서 외부 브라우저를 여는 **공식/관례 URL 스킴** 정리. (보안 우회·해킹 아님)

### 카카오톡

```javascript
var target_url = location.href;
location.href = 'kakaotalk://web/openExternal?url=' + encodeURIComponent(target_url);
```

### 라인

```javascript
var target_url = location.href;
location.href = target_url + (target_url.indexOf('?') !== -1 ? '&' : '?') + 'openExternalBrowser=1';
```

### Android 기타 인앱 → Chrome Intent

```javascript
location.href = 'intent://' + target_url.replace(/https?:\/\//i, '') +
  '#Intent;scheme=https;package=com.android.chrome;end';
```

### iOS 기타 인앱

- 강제 Safari 불가 → URL 클립보드 복사 + 안내 UI

```javascript
navigator.clipboard.writeText(location.href);
// 사용자에게 Safari에서 붙여넣기 안내
```

UA로 `kakaotalk` / `line` / 기타 inapp 분기 후 위 스킴 적용.
