## 와이어샤크 필터

```javascript
tcp contains comindev              # 내용에 특정 문자열 포함
tcp.port == 1004                   # TCP 출발지/목적지 포트
tcp.dstport == 1004                # TCP 목적지 포트
tcp.srcport == 1004                # TCP 출발지 포트
udp.port == 1004                   # UDP 출발지/목적지 포트
udp.dstport == 1004 / udp.srcport == 1004

wlan.addr == 00:0C:29:78:96:2C     # 무선랜 출발지/목적지 MAC
wlan.sa == 00:0C:29:78:96:2C       # 무선랜 출발지 MAC
wlan.da == 00:0C:29:78:96:2C       # 무선랜 목적지 MAC

eth.addr == 00:0C:29:1D:1F:0F      # 출발지/목적지 MAC
eth.src == 00:0C:29:1D:1F:0F       # 출발지 MAC
eth.dst == 00:0C:29:1D:1F:0F       # 목적지 MAC

ip.addr == 192.168.0.10            # 출발지/목적지 IP
ip.src == 192.168.0.20             # 출발지 IP

not icmp                           # icmp 제외 전체
icmp                                # icmp만(ping 확인용)
bootp                               # DHCP 패킷 (와이어샤크가 bootp로 인식)
```
