## jsDelivr (GitHub → CDN)

```
https://cdn.jsdelivr.net/gh/[user]/[repo]/[path]
```

예: `https://cdn.jsdelivr.net/gh/abcde/myrepo/lib/jquery.js`

- public GitHub repo 파일만 가능
- 캐시 만료 후 변경 반영
- 문서: [https://www.jsdelivr.com/?docs=gh](https://www.jsdelivr.com/?docs=gh)
