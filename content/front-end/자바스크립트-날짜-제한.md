## 특정 날짜 이후 페이지 접근 제한 (JS)

```javascript
var d = new Date();
var currentDate = d.getFullYear() + "년 " + (d.getMonth() + 1) + "월 " + d.getDate() + "일";
if (currentDate == '2019년 9월 16일') {
  alert("기간이 종료되었습니다.");
  location.href = "/main.do";
}
```

- JSP(`<c:if>`) 조건 안에 넣어 특정 메뉴에서만 노출 제한하는 방식으로 사용했던 패턴
